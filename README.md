# report-bug

## Installation
```bash
npm install --save report-bug
```
## Import the report-bug Library.

```js
import BugReport from 'report-bug';
```
## Usage

```js
 <BugReport user = {current_username} app_id = {current_appname} />
 ```

 ## Props

 user : _string_

 app_id : _string_
